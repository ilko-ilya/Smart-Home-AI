# 🏠 Smart Home AI (Jarvis)

**Smart Home AI** — это современное full-stack приложение умного дома. Проект демонстрирует интеграцию искусственного интеллекта (LLM) для управления устройствами, мониторинга климата и генерации контекстных рекомендаций на основе паттерна **Orchestrator**.

![Smart Home Dashboard](./screenshot.jpg)

---

## 💡 Почему проект интересен (Engineering Challenges)

Проект решает несколько нетривиальных задач, демонстрируя навыки проектирования сложных систем:

- **Интеграция LLM в backend-логику:** Использование AI не просто как чат-бота, а как функционального модуля системы
- **Парсинг и валидация AI-ответов:** Надежное извлечение JSON из ответов модели
- **Оркестрация данных:** Сбор данных из БД, сенсоров и внешних API (погода)
- **Голосовое управление:** Работа в реальном времени через Web Speech API
- **Гибридная архитектура:** Комбинация deterministic logic + AI

---

## 🚀 Ключевые возможности

### ⚙️ Гибридный подход (AI + Rule-based)

Система использует комбинированный подход:

- **Критические сценарии** ("Я дома", "Я ухожу")  
  → обрабатываются через backend-логику (без AI)

- **Гибкие команды** ("сделай комфортно")  
  → обрабатываются через LLM

💡 Это повышает надежность и снижает зависимость от AI API.

---

### 🏡 Умные сценарии (Automation)

- **"Я ухожу":**  
  включается безопасность, запускается пылесос, всё остальное выключается

- **"Я дома":**  
  отключается безопасность, включается свет в прихожей, запускается музыка (20%)

---

### 🧠 AI-ассистент (Jarvis)

- **LLM:** Llama 3.1 (Groq API)
- Анализирует:
  - температуру снаружи
  - климат внутри
  - активные устройства

- Генерирует рекомендации:
  - комфорт
  - энергосбережение

## 🔗 API Примеры

### POST /ai/voice

```json
{
  "text": "включи свет на кухне"
}
```

Response:

```
Сделано, сэр. Успешно изменено устройств: 1
```

---

### POST /ai/recommendation

Response:

```json
{
  "advice": "Сэр, на улице прохладно. Рекомендую включить обогрев."
```

## ⚙️ Запуск проекта

### 1. Клонирование

```bash
git clone https://github.com/ilko-ilya/Smart-Home-AI.git
cd smart-home-AI
```

---

### 2. Настройка .env

```
POSTGRES_USER=postgres
POSTGRES_PASSWORD=your_password
GROQ_API_KEY=your_api_key
```

---

### 3. Запуск

```bash
cd backend
docker-compose up -d --build
```

---

### 4. Открыть

```
http://localhost:3000
```
